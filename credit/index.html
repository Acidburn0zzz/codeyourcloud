Code Your Cloud could not have been possible without the following products. These were indispensable, freely available, and easy to use. In order to give the creators the credit they deserve, we have assembled this list. Give a shoutout to the developers if you have a chance.

DESIGN CREDITS

Code Your Cloud is based on Bootstrap, which improves responsiveness.

Bootstrap

"The most popular front-end framework for developing responsive, mobile first projects on the web."

To extend Bootstrap, we use FlatUI, developed by Designmodo. Available for free, FlatUI is a professional-grade theme with a modern look.

FlatUI

"Free user interface kit."

To handle our server-side code, we use Node.js. This allows us to write our client-side and server-side code in the same language.

Node.js

"Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications."

For collaboration, we used TogetherJS instead of the Google Drive Realtime API. It's easier, open-source, and has lots of features.

TogetherJS

"TogetherJS is a free, open source JavaScript library by Mozilla that adds collaboration features and tools to your website. By adding TogetherJS to your site, your users can help each other out on a website in real time!"

The editor uses Ace for syntax highlighting.

Ace

"Ace is an embeddable code editor written in JavaScript. It matches the features and performance of native editors such as Sublime, Vim and TextMate."

CodeMirror is also used, especially for the mobile version.

CodeMirror

"CodeMirror is a versatile text editor implemented in JavaScript for the browser."
