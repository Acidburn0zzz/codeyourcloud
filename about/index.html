Code Your Cloud Is On A Misson

Code Your Cloud is an in-browser collaborative code editor for Google Drive, which combines the capabilities of Google's cloud services with the functionality of CodeMirror syntax highlighting and editing.


×

Sidebar

Opens up the sidebar, which contains more functions, including a Lorem Ipusm generator, settings, and a terminal. If you don't know where something is, look here.

×

Color Picker

This color picker allows you to pick a hex color, which is automatically inserted into your code. To use, just click on it, and choose a color using its hex value or the sliders.

×

Editor

Edit your files while seeing all of the changes other users make in real time. The editor supports full syntax highlighting for almost 100 languages, as well as error detection, autocompletion, and customizable themes.

×

Toolbar

This toolbar contains common actions, so that you don't have to keep opening and closing the sidebar. Open new files, save them, chat with collaborators, and undo and redo your changes with these 5 buttons.

See how your code with look (HTML, PHP, and Markdown) and run (Javascript and Coffeescript) as you go.

Your Files Are Everywhere

With Google Drive and Code Your Cloud, you can edit your code on any computer or mobile device.

Browsers
