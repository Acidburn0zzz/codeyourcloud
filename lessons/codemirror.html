Code Mirror and Ace: tutorial and comparison

CodeMirror and Ace are the two greatest javascript-based code editors available. The choice as to which one to use is an interesting problem, and here at Code Your Cloud, we have switched back and forth several times.

1. Ace

We'll start off by learning how to make the most of the Ace editor. First, add the necessary files.

<script src= "https://codeyourcloud.com/lib/ace/src/ace.js"></script>
<script src= "https://codeyourcloud.com/lib/ace/src/modes_all.js"></script>
<script src= "https://codeyourcloud.com/lib/ace/src/themes_all.js"></script>
<script src= "https://codeyourcloud.com/ lib/ace/src/ext-language_tools.js"></script>

Then, initalize the editor.

var editor = ace.edit("editor");
//"editor" is the id of the element you are using
editor.setTheme("ace/theme/monokai");
editor.getSession().setMode("ace/mode/javascript");

Pretty simple, right?
