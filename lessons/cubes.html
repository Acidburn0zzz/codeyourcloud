However, at this point the divs are simply static, and lack any 3-dimensional orientation. To fix this, we used the rotateX, rotateY, and rotateZ properties. The x-axis runs from left to right, and a rotation around it will flip the element so that its top is now at the bottom. The y-axis runs from top to bottom, and a rotation will cause the left side of the element to finish on the right. The z-axis runs from back to front, and a rotation along it will cause the element to rotate clockwise or counter-clockwise. translate properties complete the effect.

The next step is to find out how to randomly rotate the cubes. Choosing random values is easy enough. 3 random integers (one for each cube), each multiples of 90 from -270 degrees to 270 degrees should be chosen, and then passed to a function which rotates the cubes. This should be done once every second. 

setInterval(function(){
	random_cube();
}, 1000);
function random_cube(){
	var min = -3;
	var max = 3;
	var action_random = Math.floor(Math.random() * (max - min + 1)) + min;
	var lang_random = Math.floor(Math.random() * (max - min + 1)) + min;
	var adv_random = Math.floor(Math.random() * (max - min + 1)) + min;
	
	action_random *= 90;
	lang_random *= 90;
	adv_random *= 90;
	
	rotateCube("action-cube", action_random); //to be defined
	rotateCube("lang-cube", lang_random); //to be defined
	rotateCube("adv-cube", adv_random); //to be defined
}

To animate the rotation, we would normally use jQuery's animate function. However, this function can't do this, because it can't take parameters like -webkit-transform. This is evident with an example of the animate function. 

$("#my_id").animate({
	height: "100%"
}, 500);

Therefore, we used the animate function with a made-up parameter, in this case nonexist. For each step, a function is triggered which changes the rotation of the cube. 

function rotateCube(id, goal){
	$("#" + id).animate({ nonexist: goal }, {
	    step: function(now,fx) {
	        $(this).css('-webkit-transform',"rotateX(" + now + "deg)");
	        $(this).css('transform',"rotateX(" + now + "deg)");
	        $(this).css('-moz-transform',"rotateX(" + now + "deg)");
	    },
	    duration:'slow'
	},'linear');
}
