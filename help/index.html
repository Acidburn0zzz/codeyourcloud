    Todo List

    In order to use the todo list, use the 3 keywords

    //NOTE: a note
//TODO: a todo item
//FIXME: help!

    Each of these will result in an item for the todo list. Once the item is checked off, it, along with the note, disappears. For example, if the first "NOTE" is clicked, the content will be changed to:

    //
//TODO: a todo item
//FIXME: help!

    The todo list is refreshed with every save, and can be found by clicking the "todo" menu item underneath the "code" menu, or by clicking in the sidebar.

    Google Drive Integration

    With Code Your Cloud, you can open, share, edit, upload, and save Google Drive files.

    Opening

    Click the icon, or the "open" menu item.
    Note: code your cloud CANNOT open Google Docs or Microsoft Office documents.

    Sharing

    In order to allow other people to access your files, click the "share" menu item. If you have not added people to the list of authorized collaborators, they will not be allowed to access the file.

    Creating

    Click the "new" menu item. The document will be titled by its unique file id (a very long string of characters) to prevent name collisions. The file will be created in the root folder of your Google Drive.

    Saving

    Click the "save" menu item, icon, or the save state indicator (to the right of the collaboration button).

    Uploading

    Select the "upload" menu item to upload a file to Google Drive. The file will appear in your root (main) folder.

    Collaboration

    Navigate to the document that you want to work on, and press the button.

    Running Code

    You can run code (for now, only javascript) using the button.
